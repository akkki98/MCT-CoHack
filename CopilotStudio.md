# 🧪 CoHack Lab (60 Minutes)

## Use Case: AI Training Assistant for Contoso Corp

### Scenario

**Contoso Corp**, a leading training service provider, has many **Microsoft Certified Trainers** who conduct multiple training sessions.

During training sessions, learners frequently ask questions such as:

- Where can I find the **lab instructions**?
- What are the **prerequisites for the course**?
- Can you share the **assessment link**?
- Where do we mark **attendance**?
- Can I access the **session recording**?

Currently, trainers manually:

- Share official links in the chat
- Type responses repeatedly
- Search documents during live sessions

This process can be **time-consuming**, and sometimes the answers provided are **not structured in a document format**, making it difficult for learners to reference later.

---

## 🎯 Challenge

Build an **AI Training Assistant Agent using Copilot Studio** that can:

- Answer common training-related questions.
- Provide **relevant links and documentation** instantly.
- Reduce repetitive work for trainers.
- Provide answers in a **structured format for learners**.

---

## 🧩 Lab Tasks

Participants should complete the following tasks:

### Task 1 – Create a Copilot Studio Agent
- Create a new agent called **Training Assistant Agent**
- Configure a welcome message

Example:
> Hello! I’m your Training Assistant. I can help you find labs, assessments, prerequisites, recordings, and attendance links.

---

### Task 2 – Create Knowledge Topics

Create topics for the following:

- Lab resources
- Assessment links
- Course prerequisites
- Attendance submission
- Session recordings

---

### Task 3 – Add Knowledge Sources

Add resources such as:

- Lab documentation
- Course prerequisite document
- Assessment instructions
- Attendance form link
- Recording repository link

---

### Task 4 – Build Conversation Flows

Create conversational triggers such as:

- "Lab link"
- "Assessment link"
- "Course prerequisites"
- "Attendance form"
- "Session recording"

The agent should return:

- The appropriate link
- A short explanation
- Any supporting documentation

---

### Task 5 – Test the Agent

Test the agent with example questions:

- "Where can I find the labs?"
- "Share the assessment link."
- "What are the prerequisites for this course?"
- "Where do we mark attendance?"
- "Can you share the session recording?"
